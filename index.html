SoulWorker Skill Simulator

This skill simulator require javascript turned on.
This skill simulator work on:
  • Microsoft Edge (v13 or later)
  • Mozilla Firefox (v45 or higher)
  • Google Chrome (v57 or higher)
  • Safari (v10 or higher)
For browsers which is not mentioned above: If the browser support ES6 (or ECMA 2015), it should work. Otherwise, it won't. For an example: IE11 does not support ES6, the page may show nothing if you use IE11.
